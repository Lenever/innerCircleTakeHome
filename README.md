# TakeHomeApp
A simple app showcasing a mixture of UIKit and SwiftUI components
A demo for the take home app can be found [here](https://drive.google.com/file/d/11PuHedt4NFthYPdnDUUPu9iHhqwqEGtn/view?usp=sharing)

## What is the Take Home app about?
This is a mobile application where you can see a demonstration of a project's ability to integrate UIKit and SwiftUI, manage navigation, and structure a clean, scalable iOS project..

## How does the take home app work?
- Navigates using UINaviagationController in a Coordinator class
- Navigates smoothly between UIViewControllers and SwiftUI views
- Using UIViewRepresentable to write stable UIKit components
